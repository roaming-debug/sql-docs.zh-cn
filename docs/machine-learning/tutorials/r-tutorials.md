| [适用于 SQL 开发者的数据库内 R 分析](../tutorials/r-taxi-classification-introduction.md) | 仅使用 SQL 工具来生成并部署一个完整的 R 解决方案。 重点介绍如何将解决方案移动到生产环境中。 学习如何将 R 代码包含在存储过程中，将 R 模型保存在数据库中，以及参数化调用 R 模型用于预测。 |
