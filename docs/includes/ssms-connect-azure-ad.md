> [!NOTE]
> 在 2021 年 12 月，将不再使用 MFA 通过 Azure Active Directory 对 18.6 版之前的 SSMS 版本进行身份验证。
>
> 若要继续将 MFA 与 Azure Active Directory 身份验证结合使用，需要 [SSMS 18.6 或更高版本](../ssms/download-sql-server-management-studio-ssms.md)。
