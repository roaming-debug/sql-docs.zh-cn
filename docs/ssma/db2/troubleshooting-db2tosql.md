# <a name="troubleshooting-db2tosql"></a> (DB2ToSQL 的疑难解答) 
不太常见的情况是在执行控制台脚本或命令的中间运行迁移问题。  
  
可修复的错误可能与以下内容相关：  
  
1.  函数  
  
2.  过程  
  
3.  查看  
  
4.  触发器  
  
5.  修改数据类型  
  
6.  更改项目设置  
  
通过打开通过控制台创建的项目文件，可以使用 SSMA 用户界面解决这些问题。
