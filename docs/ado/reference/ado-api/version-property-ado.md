# <a name="version-property-ado"></a>Version 属性 (ADO)
指示 ADO 版本号。  
  
## <a name="return-value"></a>返回值  
 返回一个指示版本的 **字符串** 值。  
  
## <a name="remarks"></a>备注  
 使用 **Version** 属性返回 ADO 实现的版本号。  
  
 提供程序的版本将作为 [Properties](./properties-collection-ado.md) 集合中的动态属性提供。

 [连接对象 (ADO)](./connection-object-ado.md)
