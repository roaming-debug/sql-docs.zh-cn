# <a name="concur_ss_scroll_locks-field-sqlserverresultset"></a>CONCUR_SS_SCROLL_LOCKS 字段 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  用于指定 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 有行锁的读/写乐观并发类型。  
  
## <a name="syntax"></a>语法  
  
```  
  
public static final int CONCUR_SS_SCROLL_LOCKS  
```  
  
## <a name="field-value"></a>字段值  
 一个 int 值：1009。
