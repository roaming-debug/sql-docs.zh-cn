# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>setMultiSubnetFailover 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  设置 multiSubnetFailover  连接属性的值。  
  
## <a name="syntax"></a>语法  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>参数  
 *multiSubnetFailover*  
  
 multiSubnetFailover 连接属性的新值。  
  
## <a name="remarks"></a>注解  
 若要详细了解 multiSubnetFailover 连接属性，请参阅[设置连接属性](../../../connect/jdbc/setting-the-connection-properties.md)。
