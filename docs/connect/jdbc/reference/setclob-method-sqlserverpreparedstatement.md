# <a name="setclob-method-sqlserverpreparedstatement"></a>setClob 方法 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  将指定参数设置为 java.sql.Clob 对象。  
  
## <a name="overload-list"></a>重载列表  
  
|名称|说明|  
|----------|-----------------|  
|[setClob 方法 &#40;int, java.io.Reader&#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader.md)|将指定参数设置为 java.sql.Clob 对象。|  
|[setClob 方法 &#40;int, java.sql.Clob&#41;](../../../connect/jdbc/reference/setclob-method-int-java-sql-clob.md)|将指定参数设置为给定的 java.sql.Clob 对象。|  
|[setClob 方法 &#40;int, java.io.Reader, long&#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader-long.md)|将指定参数设置为具有指定字符数长度的 java.sql.Clob 对象。|  
  
## <a name="remarks"></a>注解  
 无服务器驱动程序行为。 有关详细信息，请参阅 JSE 6 API 规范。
