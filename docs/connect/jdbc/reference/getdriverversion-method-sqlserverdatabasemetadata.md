# <a name="getdriverversion-method-sqlserverdatabasemetadata"></a>getDriverVersion 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此 JDBC 驱动程序的版本号。  
  
## <a name="syntax"></a>语法  
  
```  
  
public java.lang.String getDriverVersion()  
```  
  
## <a name="return-value"></a>返回值  
 包含 JDBC 驱动程序版本的 String。  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>备注  
 此 getDriverVersion 方法是由 java.sql.DatabaseMetaData 接口中的 getDriverVersion 方法指定的。
